## CSS

- –ö–∞—Å–∫–∞–¥ –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç –ø–æ—Ä—è–¥–æ–∫ –ø—Ä–∏–º–µ–Ω–µ–Ω–∏—è –ø—Ä–∞–≤–∏–ª: –∏—Å—Ç–æ—á–Ω–∏–∫ ‚Üí —Å–ø–µ—Ü–∏—Ñ–∏—á–Ω–æ—Å—Ç—å ‚Üí –ø–æ—Ä—è–¥–æ–∫ –æ–±—ä—è–≤–ª–µ–Ω–∏—è.

### –ú–∞–∫–µ—Ç—ã –∏ –∫–æ–º–ø–æ–Ω–æ–≤–∫–∞

### –û—Å–Ω–æ–≤–Ω—ã–µ –ø—Ä–∏–Ω—Ü–∏–ø—ã Flow Layout:

  3. **–ù–µ –≤—ã–∑—ã–≤–∞–µ—Ç –ø–µ—Ä–µ–Ω–æ—Å–∞ —Å—Ç—Ä–æ–∫–∏** –¥–æ –∏–ª–∏ –ø–æ—Å–ª–µ —Å–µ–±—è (–≤ –æ—Ç–ª–∏—á–∏–µ –æ—Ç `block`).
